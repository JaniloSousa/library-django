Sobre Este Projeto

Este projeto foi desenvolvido por volta do mês de Junho de 2023 por Janilo Paixão de Sousa afim de treinar os principais conceitos sobre o framework Django.

Neste projeto foram aplicados conceitos como:

  • Urls:
    • Criação de Urls;
    • Criação de Urls dinâmicas;
    • Sintaxe da função 'path()'.
  • Views:
    • Consultas nos models;
    • Sintaxe da função 'render()';
    • Envio de dados para um template via 'context';
    • Uso da função "get_object_or_404()";
    • Manipulação do objeto 'request';
    • Verificação se um formulário é válido;
    • Uso da função 'redirect()'.
  • Models:
    • Criação de Models;
    • Django ORM;
    • Registro dos Models na Admin do Django;
    • Tipos de dados;
    • Makemigrations e Migrate.
  • Templates:
    • Tags templates Djangos;
    • Injeção de dados dos models nos templates.
  • Forms:
    • Criação de formulários;
    • Validação de campos dos formulários;
    • Impressão de formulários nos templates;
    • Impressão de textos de erros, ajuda e 'non_field_errors'.
  • Django Messages:
    • Success Messages;
    • Error Messages.
  • Django Pagination.
